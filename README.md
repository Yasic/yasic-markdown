# yasic-markdown

自己写的 markdown->html 的css文件，需要可以拿去用，代码块中的类选择器以 pandoc 的转换标准为主。

# 用法

### 1. 引入css文件
```css
	<link rel="stylesheet" href="https://www.yasicyu.com/static/css/yasic-markdown.css">
```

### 2. 插入文本
在 __article__ 标签内插入html文本
